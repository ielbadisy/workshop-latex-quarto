\section{Introduction}

The Cox proportional hazards model \parencite{cox1972} is the
most widely used method for survival analysis.
The Kaplan-Meier estimator \parencite{kaplan1958} provides a
non-parametric estimate of the survival function.

Machine learning methods for survival analysis are reviewed
in \textcite{hastie2009}.
Model discrimination is typically assessed using
Harrell's concordance index \parencite{harrell1982}.

\section{Methods}

We follow the notation of \textcite{kalbfleisch2002}.
The Cox model specifies:

\begin{equation}
  h(t \mid \mathbf{x})
    = h_0(t) \exp(\mathbf{x}^\top \boldsymbol{\beta})
\end{equation}

Multiple citations: \parencite{cox1972, kaplan1958, harrell1982}.

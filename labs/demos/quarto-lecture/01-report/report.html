Goal

This demo shows the basic Quarto workflow: write prose, run code, label outputs, cite a source, and render the same file to HTML or PDF (Posit Software, PBC 2026).

Data

We use the built-in mtcars dataset so the example runs without downloads.

Code 
knitr::kable(head(mtcars), digits = 2)

Figure

Figure Figure 1 shows the relationship between car weight and fuel economy.

Inline Result

The dataset contains 32 cars and 11 variables.

Render

quarto render report.qmd --to html
quarto render report.qmd --to pdf
